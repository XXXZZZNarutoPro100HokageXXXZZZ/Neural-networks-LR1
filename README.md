# Neural-networks-LR1
Лабораторная работа "Введение в нейронные сети"
# Структура репозитория
1. LR1_Lite.ipynb – ноутбук для задания Lite: перебор гиперпараметров (количество нейронов, функция активации, размер батча) на наборе геометрических фигур (треугольник, круг, квадрат).
2. LR1_Pro.ipynb – ноутбук для задания Pro: бинарная классификация изображений крестиков и ноликов.
3. Pytorch LR1.ipynb – ноутбук для задания Lite: перебор гиперпараметров (количество нейронов, функция активации, размер батча) на наборе геометрических фигур (треугольник, круг, квадрат). Выполнено с использованием Pytorch

# Требования
Для запуска ноутбуков рекомендуется использовать Google Colab. Все необходимые библиотеки предустановлены в Colab.
